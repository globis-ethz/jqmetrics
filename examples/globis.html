jQMetrics

jQMetrics is a prototype implementation of an evaluation tool for web developers and designers to perform measurements along the metrics defined in our CHI 2011 paper.

The measurements are based on JavaScript for determining the DPI value of your screen and the appropriate font size.

The source code of jQMetrics is available here.
